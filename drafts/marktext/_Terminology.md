# Terminology Clarification (Our Ubiquitous Language)

## Tasks

Tasks can be called:

- tasks

- stages

- jobs (github uses "jobs")

- builds

- steps

## CI Servers

The machines that pipelines are run can be called:

- nodes

- runners

- executors

- agents

## Pipelines

Pipelines can be called:

- pipelines

- workflows

## (CD) System

The continuous deployment system can be called:

- CD platform

- CD/CD platform

- automation server

- build server
